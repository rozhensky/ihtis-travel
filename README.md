# ІХТІС Тревел — прототип нового сайту

Статичний прототип головної сторінки (ребрендинг) для туроператора **ІХТІС Тревел**.
Один файл `index.html` + фото у `assets/img/`. Без збірки й залежностей.

## Що всередині
- Верхній перемикач категорій (Airbnb-style): **Море · Екскурсії · Паломництво · Табори**
- Пошук-панель (Куди / Коли / Звідки) + стрічка-карусель і сітка напрямків
- Реальні фото країн (Wikimedia Commons — див. `assets/img/CREDITS.md`)
- Форма заявки, розділ «Агентам», світла/темна теми, адаптив

## Локальний перегляд
```bash
python3 -m http.server 8000
# відкрити http://localhost:8000
```

## Деплой на Render (trial)

**Спосіб 1 — Blueprint (рекомендовано):**
1. Залийте цю теку в репозиторій GitHub/GitLab.
2. На render.com → **New +** → **Blueprint** → підключіть репозиторій.
3. Render прочитає `render.yaml` і створить безкоштовний Static Site. → **Apply**.

**Спосіб 2 — вручну (Static Site):**
1. Залийте теку в GitHub.
2. render.com → **New +** → **Static Site** → оберіть репозиторій.
3. Build Command: *(порожньо)* · Publish Directory: `.` → **Create Static Site**.

**Спосіб 3 — Render CLI:**
```bash
# встановити CLI: https://render.com/docs/cli
render login
render deploys create <SERVICE_ID>
```

Після деплою сайт буде доступний за адресою `https://ihtis-travel.onrender.com` (або вашою).
